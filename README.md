# GameForge · 游戏企划工坊

> AI 驱动的游戏设计文档（GDD）生成器，支持 OpenAI / Claude 及兼容接口。

## 功能概览

### 核心功能
- **AI 文档生成** — 输入游戏创意，一键生成专业级游戏设计文档（GDD）
- **SSE 流式输出** — 实时查看生成过程，无需等待
- **19 种游戏类型** — 冒险、解谜、生存、策略、休闲、RPG、动作、模拟、恐怖、沙盒、竞速、体育、陪伴/养成、社交、音乐节奏、视觉小说、放置/挂机、教育、其他

### 文档编辑
- **文档可编辑** — 点击编辑按钮直接修改文档内容
- **AI 润色** — 输入修改要求，AI 对整篇文档进行润色
- **章节单独重生成** — 悬停章节标题，点击按钮仅重写该章节
- **文档追问** — 生成完成后继续追问，AI 输出修改后的完整文档
- **从已有文档导入** — 粘贴或上传已有 GDD，在此基础上修改/续写

### Prompt 系统
- **5 种提示词预设** — 默认 / 详细版 / 投资人版 / 技术侧重 / 叙事侧重
- **自定义 Prompt** — 自由编辑系统提示词，满足个性化需求
- **企划模板库** — 8 个预设模板（动作RPG、生存建造、叙事解谜、休闲合成、塔防策略、陪伴养成、恐怖探索、音乐节奏），一键填充

### 导出
- **导出 PDF** — 通过浏览器打印功能生成 PDF
- **导出 Word (.doc)** — 生成 Word 兼容格式文档
- **下载 Markdown** — 下载 .md 源文件
- **复制全部** — 一键复制文档内容
- **复制 SOLO Prompt** — 生成可直接用于 SOLO 的开发提示词

### 界面与体验
- **暗色模式** — 一键切换亮色/暗色主题，选择持久化
- **多语言支持** — 中文 / 英文切换，覆盖全部 UI 文案
- **目录导航** — 右侧 TOC 侧边栏，点击跳转 + 滚动高亮
- **历史记录** — 自动保存最近 20 条生成记录，支持查看和删除
- **响应式设计** — 适配桌面和移动端

## 快速开始

1. 用浏览器打开 `gameforge 游戏企划工坊.html`
2. 点击右上角 **设置**，配置 API（支持 OpenAI 兼容接口和 Claude）
3. 输入游戏创意，选择类型、目标玩家、开发周期
4. 点击 **生成企划**，等待 AI 生成文档
5. 使用追问、编辑、润色等功能完善文档
6. 导出为 PDF / Word / Markdown

## API 配置

### OpenAI 兼容接口
| 字段 | 说明 |
|------|------|
| API 地址 | 完整的 Base URL，如 `https://api.openai.com/v1` |
| API Key | 你的 API 密钥 |
| 模型 | 模型 ID，如 `gpt-4o`、`deepseek-chat`、`glm-4-flash` |

### Claude
| 字段 | 说明 |
|------|------|
| API Key | Anthropic API 密钥 |
| 模型 | 模型 ID，如 `claude-sonnet-4-20250514` |

> **注意**：部分 API 服务可能不支持浏览器直接调用（CORS 限制）。如果遇到连接失败，请确认 API 服务是否允许跨域请求。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + Enter` | 生成文档 |
| `Ctrl/Cmd + S` | 退出编辑模式 |
| `Esc` | 关闭弹窗 |

## 技术说明

- **单文件应用** — 纯 HTML + CSS + JavaScript，无需构建工具
- **无外部依赖** — 不依赖任何第三方库或 CDN
- **数据存储** — 所有配置和历史记录保存在浏览器 localStorage 中
- **Markdown 渲染** — 内置轻量级 Markdown 渲染器，支持标题、表格、列表、引用、代码等
